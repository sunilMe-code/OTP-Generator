🔐 OTP Generator

A simple and responsive OTP (One-Time Password) Generator built using HTML, CSS, and Vanilla JavaScript. This project generates a random OTP that can be used for basic verification purposes.

The application allows users to generate a new OTP instantly with a button click. It demonstrates how to use JavaScript random functions, DOM manipulation, and event handling to create interactive web features.

This project is ideal for beginners who want to learn how OTP systems work at the frontend level. It does not include backend verification but focuses on UI and logic implementation.

✨ Features
🔢 Generate random OTP
🔄 Regenerate OTP on click
📋 Copy OTP functionality (optional)
🎨 Clean and modern UI
📱 Fully responsive design

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling
JavaScript (ES6) – Logic
